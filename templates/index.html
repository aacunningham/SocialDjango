About This Site

This site was built in order to expand my knowledge of Django, a python web framework, as well as demonstrate my skills to those interested. This project broadened my knowledge of web development, stepping me into the MVC (Model, View, Controller) model of web sites, as well as allowing me to work on my javascript and CSS.

There are several frameworks/libraries that this project uses to function, several of them being very vital to modern web development and design. The first, as mentioned, is Django, an MVC framework "designed for perfectionists with deadlines". This is a very popular framework for Python, allowing for very quick building and prototyping of web applications, making it easy for someone to create a functional site with little to code. Django is designed to handle the backend of a website, or the server functions, and process url routing, database querying, and template rendering for dynamic websites.

While Django handles the backend, though, I needed a framework that could help with frontend. That's where Bootstrap comes in (also known as Twitter Bootstrap). Bootstrap is a CSS framework that helps create websites that beautiful to look at while also being responsive to change. Bootstrap, like many other web frameworks, uses a grid system that allows for content to be responsive to the size of the screen viewing it. While achieving this functionality would be possible without the framework, Bootstrap makes getting a project started a lot faster, allowing me to worry less about design and more on content.

Finally, while backend and design are important, interactive sites are what keep people interested, and so I practiced my use of JavaScript, and through it, the common JavaScript library jQuery. With both of these combined, I was able to implement posts on this site that can be added and deleted dynamically, without reloading the full page. This took some time to fully understand, not to mention getting the hang of AJAX requests, but I feel confident that the end result was worth the effort.

More About This Site

This site is somewhat functional, though maybe not entirely "social". It allows for people to create a user, create posts on their account, login, logout, and delete their account. Authorized (logged in) users may search for others, and view their profile pages, which includes their post history and their bio. Posts and user bio allow for light html, such as <strong> and <em>, and strips out any dangerous code (such as script tags to prevent any javascript injection). There is also the ability to change your password and change your bio. This site should also be easy to view on phones, though it may need some tweaking.

Future plans include adding a form of friendship to the site, using M2M modeling. Afterwords, adding comments to posts as well as posting on other peoples pages will be worked into the site. Then, I would like to look into more photos for each user, along the lines of Facebook's photo system. Finally, a way to customize the profile page would be very interesting to look into.
